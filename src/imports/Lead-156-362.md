# Zenite Reports - "Todos Relatorios" (All Reports)

Documentacao extraida do componente Figma `Lead-156-362.tsx`.

---

## 1. Visao Geral

Tela de listagem de relatorios no ecossistema Zenite. Apresenta uma sidebar de filtros rapidos a esquerda, uma tabela de relatorios no centro/direita, e controles de acao no topo. Suporta dois modos de visualizacao: **Tabela** (ativo) e **Pasta**.

**Dimensoes do container:** 1093 x 791px, fundo branco, border-radius 15px.

## 9. Interacoes Esperadas

1. **Tabs de filtro** — clique alterna estado ativo (background `#DDE3EC`)
2. **Segmented control** — alterna entre visualizacao Tabela e Pasta
3. **Checkboxes** — selecao individual de relatorios (multi-select)
4. **Nome do Relatorio** — link clicavel (cor `#07ABDE`), abre o relatorio
5. **Nome do Criador** — link clicavel (cor `#07ABDE`), abre perfil
6. **Botoes de acao** — hover mostra gradiente radial, click para acao
7. **Filtros Rapidos** — expande/colapsa painel de filtros

## 10. Notas de Implementacao

- Layout usa posicionamento **absoluto** (pixel-perfect do Figma); para producao, migrar para Flexbox/Grid responsivo.
- Icones: usar **Phosphor Icons** (`@phosphor-icons/react`) — `import { FunnelSimple, FolderSimple, ... } from "@phosphor-icons/react"`. Referencia completa em https://phosphoricons.com.
- `backdrop-blur` e usado em checkbox e botao de filtros — garantir suporte do navegador.
- Linhas da tabela usam `display: contents` com posicionamento absoluto por inset percentual — reestruturar como `<table>` ou grid para producao.
- O avatar "3D Avatars / 29" e uma imagem raster importada via `figma:asset` — considerar placeholder ou integracao com sistema de avatares real.
